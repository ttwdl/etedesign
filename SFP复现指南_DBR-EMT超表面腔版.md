# SFP 复现改造指南:介质 DBR + 等效介质(EMT)超表面腔 + TMM 端到端联合设计

**对标论文**:S. Feng, Z. Wang, X. Cheng, X. Dun, "Superposition Fabry–Perot filter array for a computational hyperspectral camera," *Opt. Lett.* **48**(5), 1156 (2023), DOI: 10.1364/OL.479622(含 Supplement 1)
**改造要点(v1.1)**:上下反射镜由 Ag 换为介质 DBR;腔层为"你的固定厚度亚波长纳米柱图案 + 光刻胶充填固化",用等效介质理论(EMT)将填充因子 f 映射为 n_eff(f, λ),逐像素调 f 而非腔长;通道数 M=16(4×4 探测器像素),每滤波器为单一均匀图案区域;端到端框架、TMM 编码层、解码器、数据与训练配置与原文逐项对齐。

> **v1.1 修订记录(按三点更正)**:① 腔体 = 你的纳米柱结构 + 光刻胶充填后固化,其上直接沉积顶 DBR(免 SiO₂ 回填与 CMP,但引入残留胶层 t_r 与热预算约束,见 R4/R7);② 每滤波器只有**一个**图案区域,T_l = T(f_l),取消子区叠加;③ 通道数 M 由 4(2×2)改为 **16(4×4 探测器像素)**,解码器输入维随之 4→16。

## 0. 核心思想(一段话)

原文以 Ag/SiO₂/Ag 三层膜为编码滤波器,以灰度电子束光刻(G-EBL)产生的空间变腔长 h₂ 作为逐像素调谐自由度。你的版本:两面 Ag 镜 → 介质 DBR(全片同炉沉积,所有像素共享);腔层 → "你的高折射率纳米柱结构 + 光刻胶充填后固化"的亚波长图案层(胶面之上直接沉积顶 DBR),腔厚 h_c 全片固定(单次刻蚀深度),逐像素只改柱直径 D(即填充因子 f = πD²/4P²)。f 经 EMT 映射为 n_eff(f, λ),n_eff 进入与原文完全相同的 TMM 特征矩阵编码层,再接原文同款全连接解码网络端到端联合训练。梯度链:

**Loss → 解码器 → I_l → T_{l,λ}(TMM)→ n_eff(EMT)→ f → D**

全程解析可微,无需建立"结构→光谱"训练数据集——这正是原文引入 TMM 编码层(替代 Song et al. 的网络代理编码层)的核心优势,你的版本只是在 TMM 前面再串一个同样解析可微的 EMT 节点。

**相对原文的三个预期增益**:(1)全介质无 Ag 吸收,带内峰值透过率可从原文的 ~0.4 提升到 0.7 以上(即论文标题所指"高效");(2)二值光刻 + 单一刻蚀深度取代灰度 EBL,免去灰度剂量标定,更利于大规模制造;(3)图案化腔的 n_eff(胶充填 + TiO₂ 柱约 1.66–1.98)高于原文 SiO₂ 腔(1.46),FP 角度不敏感性(∝ 腔内高折射率)持平或略优。

## 2. 物理建模指南

### 2.1 结构与默认参数

层序(光从空气侧入射):

```
空气 / [H L]×p_top / 残留胶层(n_resist, t_r) / 腔层(n_eff(f,λ), h_c) / [L H]×p_bot / 熔石英衬底
```

其中 t_r 为旋涂固化后**柱顶之上的均匀残留胶厚**(自平坦化的必然产物,若工艺回刻/灰化到柱顶则 t_r=0),必须作为独立 TMM 层建模,否则腔的有效光程被系统性低估。

| 参数 | 默认值 | 说明与调参范围 |
|---|---|---|
| 波段 | 400–700 nm,151 点 @2 nm | 与原文一致 |
| H 层 | TiO₂,h_H = λ₀/(4n_H) ≈ 56 nm(λ₀=530 nm,n_H≈2.35) | 可换 Si₃N₄/Ta₂O₅;顶 DBR 须低温工艺(R7) |
| L 层 | SiO₂,h_L = λ₀/(4n_L) ≈ 91 nm(n_L≈1.46) | — |
| DBR 对数 p | 2(可配 3) | **越少越好**:中等反射率 → 峰宽、多峰、高通量,更利于压缩编码(见 §2.4) |
| 柱材料 | **TiO₂(n≈2.35,推荐)** | 备选 Si₃N₄(n≈2.02,胶充填后折射率对比度偏低)、a-Si:H(蓝端吸收,慎用) |
| 充填材料 | **固化光刻胶**(默认 SU-8,n≈1.58–1.60,Cauchy 色散) | 以椭偏实测为准(假设 A10);兼作平坦化层,其上直接沉积顶 DBR |
| 残留胶层 t_r | 初值 50 nm,∈ [0, 150] nm | 全局共享;可训练或旋涂后椭偏实测固定(假设 A11) |
| 周期 P | 180 nm,固定不训练 | 亚波长条件见 §2.2 |
| 柱直径 D | ∈ [60, P−g_min] nm,g_min 默认 40(建议扫 {40, 60, 80}) | 下限=光刻最小特征;注意胶充填深宽比 h_c/g_min ≤ ~8 为稳妥(R4) |
| 填充因子 f | [0.087, 0.475](g_min=40;g_min=60 时上限 0.349) | 每滤波器 1 个、共 16 个可训练,sigmoid 界定 |
| 腔厚 h_c | 初值 600 nm,∈ [300, 1200] nm | 全局共享可训练 |

**调谐范围算例**(供 sanity check,端到端训练会自行寻优):光刻胶充填(n_f≈1.58)下,TiO₂ 柱 f 从 0.087 到 0.475 时 n_eff 从 1.66 到 1.98(体积平均式);Si₃N₄ 柱仅 1.62 → 1.80,对比度明显缩水——这正是 v1.1 把默认柱材改为 TiO₂ 的原因(胶的折射率高于 SiO₂,吃掉了低端对比度)。若因深宽比把最小间隙放宽到 60 nm(f_max=0.349),TiO₂ 柱仍有 1.66 → 1.89。取 h_c = 600 nm、TiO₂ 柱:光学厚度 n_eff·h_c ≈ 996–1188 nm,忽略 DBR 相位时 FP 级次 λ_m = 2n_eff·h_c/m 给出:m=3 落在 664–792 nm、m=4 落在 498–594 nm、m=5 落在 398–475 nm——可见光内同时存在约 3 个可随 f 移动的谐振峰,叠加 DBR 停带边与侧带纹波,**单个滤波器**的透过谱即为"多峰宽带谱";16 个不同 f 的滤波器把峰位铺满全带,以通道数弥补取消子区叠加带来的单谱丰富度损失。

### 2.2 EMT 模块

填充因子(正方晶格圆柱,来自你上传学位论文式 2.10):

```
f = πD² / (4P²)
```

**闭式默认**(论文式 2.15–2.16,体积平均/Wiener 上界):

```
n_eff(f, λ) = sqrt[ (1−f)·n_fill(λ)² + f·n_pillar(λ)² ]
```

**备选 MG-2D**(E 垂直柱轴的二维 Maxwell-Garnett,低 f 段对面内电场物理上更贴切):

```
ε_eff = ε_f · [(ε_p+ε_f) + f(ε_p−ε_f)] / [(ε_p+ε_f) − f(ε_p−ε_f)]
```

三点必须写进代码注释与论文方法节的物理事实:

1. **有效性条件(腔内无衍射级)**:P < λ_min/n_eff,max。默认 P=180 nm、λ_min=400 nm ⇒ 要求 n_eff,max < 2.22,胶充填下 TiO₂ 柱(1.98)与 Si₃N₄ 柱(1.80)方案均满足。相邻 TiO₂ DBR 高折层内 400/2.35≈170 nm 略小于 P,但该层为深亚波长薄层、高阶为倏逝波,是否引入可观误差交由 §4.2 RCWA 裁决。n_fill 取固化胶的实测 Cauchy 色散(SU-8 默认 n(λ) ≈ 1.566 + 0.00796/λ² + 0.00014/λ⁴,λ 单位 μm);部分胶在 400–450 nm 有弱吸收,必要时给 n_fill 加小虚部并同样进 TMM。
2. **偏振与对称性**:正方晶格 + 圆柱具 C4 对称,正入射偏振无关(论文式 2.16 后的论述)。斜入射时等效介质为单轴各向异性(光轴=柱轴):E 沿柱轴分量的 ε 严格等于体积平均,面内分量更接近 MG;≤5° 时 cos5°≈0.996,各向异性影响可忽略,用各向同性 n_eff 近似即可,>8° 需换双轴 TMM。
3. **精度阶梯与关键警示**:谐振波长对 n_eff 误差有放大效应,δλ/λ ≈ (δn/n)×(腔占总光程比)。闭式 EMT 若有 2% 误差,峰位可错 5–8 nm,足以让训练所得感知矩阵失真。因此采用三级阶梯:**闭式 EMT(初始化 + 梯度骨干)→ RCWA 标定的 n_eff*(f,λ) 查找表(可微样条,阶段 4 必做)→ 可选 torcwa 在环精调(阶段 6)**。原文致谢中也提到 RCWA 仿真指导,说明作者同样以 RCWA 复核,本复现将其制度化。

### 2.3 可微 TMM(与原文式 (1)–(4) 严格一致)

测量方程(原文式 1):

```
I_l = Σ_λ S_λ · T_{l,λ}(α) · D_λ ,   l = 1, …, M
```

特征矩阵(原文式 2,Born & Wolf):对层 j = 1…k,

```
δ_j = (2π/λ) · n_j(λ) · h_j · cosθ_j ,   cosθ_j = sqrt( 1 − (n₀·sinα / n_j(λ))² )
M_j = [ [cosδ_j ,  (i/η_j)·sinδ_j ],
        [i·η_j·sinδ_j ,  cosδ_j  ] ]
[B; C]ᵀ = ( M_1 · M_2 · … · M_k ) · [1; η_sub]ᵀ
```

导纳(原文式 3):s 偏振 η_j = n_j·cosθ_j;p 偏振 η_j = n_j/cosθ_j。透过率(原文式 4):

```
T(λ, α, pol) = 4·η₀·Re(η_sub) / |η₀·B + C|² ,   非偏振取 T = (T_s + T_p)/2
```

实现要点:全复数张量(训练 complex64、测试 complex128);批处理维度 [结构数, 角度数, 波长数],矩阵连乘沿层维做,禁止对波长写 Python 循环;吸收介质(EXP-1 的 Ag)复折射率开方取 Im(n·cosθ) ≥ 0 分支;单元测试对照 S. Byrnes 的 `tmm` 包(<1e-6)并检查无吸收时 R+T=1(<1e-8)。全介质版无复折射率分支问题,数值上比原文更干净。

### 2.4 设计空间预算与编码哲学

计算光谱编码要的**不是单个窄峰,而是彼此互异、覆盖全带的宽带多峰谱**(原文用 tor 指标量化,SFP 的 tor=14.75% 介于纯 FP 的 10.88% 与十层膜 TF 的 23.29% 之间)。因此:DBR 对数取 2–3 对即可(反射率 ~0.7–0.9,中等精细度),峰更宽、通量更高;停带外的 DBR 侧带纹波本来就在 TMM 前向模型里,同样是有效编码特征,不必视为杂散;调谐靠"f 移动多个级次的峰位 + h_c 全局定级次布局 + 16 个通道的 f 全带铺开"共同完成。

一个必须直视的事实:**单区域滤波器在结构类别上属于原文表现最差的"FP resonance cavity"基线**(原文 4 通道下其 tor 仅 10.88%、MSE 1.58×10⁻³)。本版靠三点补偿:① 通道数 4→16(原文补充表 S4:SFP 从 7.42→1.51×10⁻⁴,通道数增益显著);② 低精细度 DBR 的多级次 + 侧带宽谱,单谱信息量远高于金属高精细度单峰 FP;③ 端到端训练会自动把 16 个 f 摆到使感知矩阵最互异的位置。若训练后 tor 仍明显偏低,优先动作依次为:释放 DBR 层厚全局微调 → 放宽 g_min 换取更大 f 上限(权衡 R4 深宽比)→ **无成本回引每滤波器 2×2 子区**(同一次二值光刻,仅版图变化,是本结构相对灰度 EBL 的独有退路)。

## 3. 端到端框架(与原文逐项对齐)

### 3.1 编码层

可训练参数**严格分两组**,对应真实工艺的参数共享结构——这是本改造建模的核心纪律:

- **全局共享**(整片一次沉积/一次刻蚀/一次旋涂决定,所有像素相同):h_c(sigmoid 界定于 [300,1200] nm,初值 600);残留胶厚 t_r(∈[0,150] nm,初值 50,开关 `train_tr` 控制可训练或按实测固定);可选开关 `train_dbr`:各 DBR 层厚在四分之一波 ±20% 内微调。
- **逐滤波器**:ρ_l → f_l = f_min + (f_max−f_min)·sigmoid(ρ_l),l = 1..16(每滤波器单一区域),初始化 U(f_min, f_max)(对应原文"界内均匀随机初始化"的做法)。

前向伪代码:

```python
def forward(S, alpha):                      # S: [B, 151] 归一化光谱
    f     = f_min + (f_max - f_min) * sigmoid(rho)        # [M=16]
    n_cav = emt(f, wl)                                    # [16, 151]  可微
    stack = assemble(dbr_top, (n_resist, t_r),
                     (n_cav, h_c), dbr_bot)               # 折射率/厚度张量
    T     = tmm(stack, wl, alpha).mean(pol)               # [16, 151] 单区域,无子区平均
    I     = (S * D_lambda) @ T.T                          # [B, 16]  原文式(1)
    return I + gaussian_noise(sigma ~ U(0, 0.001))
```

入射角:训练时每样本 α ~ U(0°, 5°)(原文只说明编码层含 α、系统工作角 5°,采样方式未公开,此为假设 A5;备选:3 节点 Gauss–Legendre 锥内求积取平均);评估固定在 {0, 0.5, 2, 5}° 并扩展 {8, 10}°。D_λ 默认全 1(假设 A4,可载入硅 QE 曲线 CSV)。

### 3.2 解码器(按原文架构字符串,输入维 = 通道数 M)

原文为 `151-Opticalmodel-4-FC-LR-500-FC-LR-151`;本版通道数改 16,即 `151-Opticalmodel-16-FC-LR-500-FC-LR-151`:

```
Linear(M → 500) → LeakyReLU → Linear(500 → 151) → LeakyReLU    # EXP-1: M=4;EXP-2: M=16
```

不加 BN/Dropout。复现成功后再考虑升级(更深 MLP / 1D-CNN / 谱注意力),升级实验单独编号,不与复现混算。

### 3.3 数据与训练(对齐原文 Supplement §3)

CAVE(32 景)+ NTIRE 2020,均为 400–700 nm、31 通道 @10 nm;逐像素抽光谱,三次样条插值 31→151 通道 @2 nm;每条光谱除以自身最大值归一(假设 A6,原文只说 normalized);随机抽 1×10⁶ 条训练、1×10⁵ 条测试,**按场景划分**避免像素级泄漏。优化器 SGD + momentum=0.9(假设 A2),lr = 1×10⁻³,300 epochs,batch = 2048(假设 A1),loss = **MSE**(原文补充表 S2 实测 MSE 优于 RMSE/RMAE)。测量噪声:σ 每样本 ~U(0, 0.001) 的加性高斯(施加于归一化 I 上,假设 A3)。参考时长:原文 RTX 3060 上约 3 小时/300 epochs。多种子重启 ≥3 次取最优(结构参数优化存在局部极小)。

### 3.4 制造约束与鲁棒性

- **界约束**:D ∈ [60, P−g_min] nm 已通过 sigmoid 重参数化内建(最小特征 + 最小间隙,g_min 默认 40、建议扫 {40,60,80})。
- **量化**:收敛后把 D 吸附到光刻栅格(EBL 1–2 nm / DUV ~5 nm),两条路线任选:(a)量化后冻结编码器、仅微调解码器 20–50 epochs;(b)直通估计器(STE)联合微调。验收:量化致 MSE 退化 <5%。
- **制造误差行**(对齐原文 Table 1 末行 "SFP with fabrication err",±3 nm):MC 100 次,D ±3 nm、h_c ±2 nm、t_r ±5 nm、各折射率 ×(1±0.5%),报告均值。原文 SFP 该行 0° 下 MSE 仅从 7.03 升至 7.19×10⁻⁴(退化 2.3%),作为对标线。
- **可选鲁棒训练**:训练中对 D 注入 ±3 nm 扰动(原文未做,属增强项,单独消融)。

## 4. 验证与评价体系

### 4.1 单元级测试(pytest)

`tmm_torch` 对照 `tmm` 包 20 组随机膜系(含 Ag 吸收层、斜入射 s/p),double 精度 max|ΔT|、max|ΔR| < 1e-6;无吸收结构 R+T−1 < 1e-8;`torch.autograd.gradcheck` 对 h、f 通过(complex128);EMT 边界:f=0 ⇒ n_eff=n_fill,f=1 ⇒ n_eff=n_pillar;前向+反向速度:16 结构 × 151λ × 4 角 < 50 ms(GPU)。

### 4.2 RCWA 交叉验证协议(阶段 4,必做)

指标:MSE、PSNR、SAM,外加原文补充 §10 的谱调控能力指标
